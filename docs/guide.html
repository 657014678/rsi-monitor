一、为什么选红利低波+价值100？

🧬 红利低波（512890 · 东证红利低波指数）

  • 选股逻辑：从全市场选股息率高+波动率低的100只股票
  • 每年调仓两次，天然"高抛低吸"——涨多了股息率下降自动卖出，跌多了股息率上升自动买入
  • 指数自2005年以来年化收益约 12-15%，最大回撤远小于沪深300
  • "收息+赚价差"双轮驱动，熊市靠分红扛住，牛市靠估值修复吃肉

💎 价值100（159263 · 国证价值100指数）

  • 选股逻辑：从全市场选估值低+基本面好的100只股票
  • 价值因子长期有效：低PE/PB组合在A股历史表现显著优于高估值组合
  • 与红利低波互补——红利低波偏大盘稳健，价值100覆盖更多中盘价值股

🤝 为什么两个一起配？

  • 分散 单一策略有风格轮动期，双策略平滑波动
  • 共振 两者在市场极度低估时同时发出加仓信号，集中火力
  • 对冲 大盘价值vs中盘价值，风格不完全同步
二、RSI21 是什么？为什么用21？

📐 RSI 原理（Relative Strength Index）

RSI 衡量一段时间内涨的力量 vs 跌的力量，取值0-100：

RSI含义市场情绪
>70涨的力度远超跌过度乐观/超买
50-70涨的力度略强偏强
30-50跌的力度略强偏弱
<30跌的力度远超涨过度悲观/超卖

🤔 为什么是RSI21而不是RSI14？

  • RSI14是传统默认值，偏短线交易，信号频繁
  • RSI21约等于1个月交易日的周期，与月度定投节奏匹配
  • 更长周期 = 更少噪音 = 更少假信号 = 更少冲动操作
  • 配合MA250长周期均线，形成"月级择时+年级趋势"的双重保护
三、MA250 是什么？为什么用它分区？

📏 MA250 = 250日均线 ≈ 一年均线

  • 250个交易日≈一年，MA250反映过去一年的平均成本
  • 价格在MA250上方 = 强势区 → 大多数持仓者盈利，市场偏乐观
  • 价格在MA250下方 = 弱势区 → 大多数持仓者亏损，市场偏悲观

🎯 分区的核心意义

  • 强势区 可以做减仓+回补（市场给了你获利了结的机会）
  • 弱势区 只买不卖（亏着卖是最愚蠢的事，浮亏不是真亏）
  • 分区不是为了预测涨跌，而是决定你能做什么操作
四、策略设计的底层逻辑

🧠 为什么"低位多买"比"高位少买"重要？

数学原理：同样涨10%，从1元涨到1.1元你赚10%，从0.8元涨到0.88元你也赚10%——但你在0.8元时多买了25%的份额，所以实际收益是12.5%。

低位多买的复利效应，远大于高位少买省下的那点钱。

🛡️ 为什么弱势区"永远不减仓"？

  • 弱势区减仓 = 浮亏变现为真亏
  • 红利低波和价值100的成分股都是盈利稳定的好公司，下跌是暂时的
  • 历史数据：512890自2018年成立，每次跌破MA250后最终都涨回来了
  • 你卖出的那个价格，很可能是别人眼中"打折的好机会"

⏳ 为什么30天冷却期？

  • 防止"减了又买、买了又减"的反复操作
  • 每次操作都有摩擦成本（赎回费、申购费）
  • 冷却期强迫你等待，用时间换空间
  • 30天≈1个月，与定投节奏对齐，不会增加操作复杂度

📅 为什么每月只操作1次（21日）？

  • 定投的本质是"纪律"，不是"择时"
  • 每月1次操作，消除日内情绪波动的影响
  • 21日选在月中，避开月初月末的资金面波动
  • 操作越少，犯错的概率越低
五、心理按摩室
🧘

😰 "我亏了好多钱，要不要跑？"

你看到的"亏损"是浮亏，不是真亏。只要你不在弱势区卖出，这个数字没有任何意义。

红利低波ETF的成分股每年分红4-5%，即使价格不涨，你的份额也在积累越来越多的分红权。时间的玫瑰会绽放。

😱 "大盘暴跌，我慌了"

暴跌恰恰是策略发挥威力的时候：

  • RSI跌破40 → 你自动加仓到1.5-2份
  • 跌破MA250 → 弱势区铁律保护你，不会让你卖出
  • 别人恐惧时你贪婪——不是口号，是写在策略里的纪律

问自己：公司的基本面变了吗？分红取消了吗？都没有的话，价格下跌只是在打折。

😤 "涨了这么多，我要不要先卖了锁利？"

如果你在强势区、RSI≥65、不在冷却期——策略已经告诉你减仓了，按策略执行。

如果你不满足减仓条件——说明市场还没到过热，耐心持有。提前下车是最常见的踏空原因。

😩 "为什么我的策略跑不赢别人？"

你看到的"别人"是幸存者偏差。赚了的人会晒，亏了的人沉默。

你的策略目标不是"跑赢所有人"，而是用最小的心理压力获得合理的长期回报。年化5-8%，持续20年，结果是惊人的。

🥱 "定投太无聊了，我想做点刺激的"

无聊 = 纪律 = 赚钱。刺激 = 情绪 = 亏钱。

如果你真的手痒，拿总资金的5%去做短线，亏光了也无所谓。剩下的95%严格按策略走。让你的"赌性"和"纪律"井水不犯河水。

😴 "我坚持不下去了"

每个人都会经历这个时刻。但你想想：

  • 你在弱势区坚持定投，每一次买入都在收集更便宜的筹码
  • 市场终会回归均值，这是被几百年金融史反复验证的铁律
  • 放弃的那一刻，往往就是黎明前最暗的时刻
"股市是把钱从没耐心的人转移到有耐心的人手里的工具。"
—— 巴菲特
六、极端行情应对手册

📉 单日暴跌 >5%

  • 不做 当天不要任何操作，关闭行情软件
  • 等待 等到21日操作日，看RSI和MA250再决策
  • 暴跌后的第二天往往有反弹，冲动操作容易两面挨耳光

📉 连续下跌一周以上

  • 检查是否进入弱势区 → 如果是，按定投表加倍投入
  • 检查RSI是否跌破40 → 如果是，下次操作日加码到1.5-2份
  • 回忆：512890在2022年4月、2022年10月、2024年1月都跌破MA250，每次都在3-6个月内涨回

🔥 连续暴涨，FOMO严重

  • 检查RSI → 如果≥65，下次操作日按策略减仓15%-30%
  • 如果RSI在50-64区间，什么都不做，继续持有
  • 不要在非操作日追涨——你的策略里没有"追涨"这个选项

🌊 全市场恐慌（千股跌停级别）

  • 这是十年一遇的加仓机会
  • RSI很可能跌到40以下，下次操作日直接2份
  • 红利低波的成分股是最抗跌的一批公司，恐慌中它们跌得最少、弹得最快
  • 历史不会重复，但会押韵——每次大恐慌后都是丰厚回报
