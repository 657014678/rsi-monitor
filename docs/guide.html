一、策略总览

🎯 投资目标

  • 长期持有，低回撤，复利增长
  • 年化目标：8-15%（不追求暴利）
  • 最大回撤控制：≤20%
  • 操作频率：每月仅1次（21日执行）

📊 三标的配置

  • A股 红利低波 40%（012708 · 东证红利低波）
  • A股 价值100 40%（025497 · 国证价值100）

A股双价值打底，美股科技增强，低相关性分散风险。

🧠 核心逻辑（一句话）

低位多买，高位少买，强势区减仓，弱势区死扛。

用数学纪律代替情绪决策，用时间换空间。

二、A股策略：RSI21 + MA250
适用：红利低波 · 价值100

📐 指标说明

指标含义作用
RSI2121日相对强弱指数判断"超买/超卖"
MA250250日均线（≈年线）判断"强势/弱势区"

为什么是RSI21？与月度定投节奏匹配，过滤短期噪音。

🗺️ 市场分区规则

分区条件可执行操作
强势区价格 ≥ MA250定投 + 减仓 + 回补
弱势区价格 < MA250只定投，不减仓，不回补

⚡ 铁则：弱势区不减仓

  • 浮亏卖出 = 真亏 + 确认亏损
  • 红利/价值100的成分股都是盈利稳定的好公司，下跌是暂时的
  • 弱势区只做一件事：按表定投，收集便宜筹码

三、美股策略：PE分位 + 回撤
适用：纳指100（159941）

📐 为什么纳指100不用RSI+MA250？

  • 纳指100长期牛市，MA250几乎永远在价格下方 → 弱势区信号极少
  • 美股科技股波动大，RSI频繁超买超卖 → 信号噪声太大
  • PE分位直接看"贵不贵"，回撤直接看"打折没" → 更直观，更适合纳指

📊 指标说明

指标含义数据来源
PE分位当前PE(TTM)在近7年中的排名百分位乐咕乐股
高点回撤(当前价-阶段最高价)/阶段最高价新浪财经

🗺️ PE分位分区

PE分位估值状态定投动作
<30%极度低估2.5份封顶
30–49%低估1.8份
50–69%合理1.0份（基准）
70–84%高估0.5份
≥85%极度高估暂停定投

🌊 回撤分区

回撤幅度分区对操作的影响
<8%正常波动按PE分位正常执行
8–15%中级回调可适度加码
>15%深度回调永不减仓！可大幅加码

⚡ 铁则：深度回调（回撤>15%）永不减仓

  • 深度回调时即使PE高估，也只暂停定投，绝不卖出
  • 历史数据：纳指100深度回调后6-12个月内多能收复失地

📉 纳指减仓规则

触发条件（全部满足）：
  1. PE分位 ≥ 70%
  2. 回撤 < 8%（正常波动，非深度回调）
  3. 不在60天冷却期内
PE分位减仓比例
70–84%减仓 15%
≥85%减仓 30%

纳指冷却期60天（比A股长），因为美股波动大，避免频繁操作。

📈 纳指回补规则

触发条件（全部满足）：
  1. PE分位 < 70%（退出高估区）
  2. 回撤 < 15%（不在深度回调中）
  3. 之前有减仓 + 减仓日期在60天冷却期外
PE分位回补比例
40–50%接回已减仓的 2/3
<40%接回全部剩余
四、执行方法（每月21日）
1
打开 价值投资控制台（本网页）
2
查看 三标的估值横条，了解当前市场状态
3
展开每个标的卡片，查看 定投建议（几份）
4
如有 减仓/回补建议，确认不在冷却期内再执行
5
按建议金额执行申购/赎回（在券商APP操作）
6
如有减仓操作，在网页 持仓记录 中登记日期和比例

⏰ 为什么是21日？

  • 避开月初（资金面紧张）和月末（机构结算）
  • 与工资现金流匹配（多数人到账时间在15-20日）
  • 遇节假日顺延到下一个工作日

📊 "一份"是多少钱？

由你自己决定。建议：

  • 先计算每月可用于定投的闲钱总额
  • 三标的各40%/40%/20%，算出每个标的的基准月投额
  • 这个基准月投额 = 1份
  • 例如：月投总额6000元 → 红利低波2400元(1份) → RSI40时投入2份=4800元

六、铁则汇总（必须牢记）

⚡ 铁则1：3年以上闲钱

  • 短期要用的钱不进市场
  • 定投的本质是用时间换空间，需要足够长的时间维度

⚡ 铁则2：无杠杆，不融资

  • 不用融资融券，不做杠杆ETF
  • 复利的前提是"还在场上"，杠杆会让你被迫离场

⚡ 铁则3：每月只操作1次

  • 只在每月21日操作（遇节假日顺延）
  • 非操作日不看净值，不操作，不后悔

⚡ 铁则4：弱势区（价格<MA250）不减仓

  • 浮亏不是真亏，卖出才是
  • 弱势区只做一件事：按表定投

⚡ 铁则5：深度回调（回撤>15%）永不减仓

  • 适用于纳指100
  • 即使PE高估，深度回调时也只暂停定投，不卖出

⚡ 铁则6：冷却期内不重复操作

  • A股减仓后30天内不回补（回补时重新计时）
  • 纳指减仓后60天内不回补（美股波动大，冷却期更长）
  • 冷却期在网页"持仓记录"中自动计算，无需记忆

⚡ 铁则7：不追涨，不抄底，按表操作

  • 策略里没有"追涨"选项，也没有"抄底"选项
  • 所有的买入/卖出都按RSI/PE分位的既定梯度执行
  • 你的情绪是最大的敌人，策略是你的盾牌
七、常见问题

❓ 红利低波没有ETF，怎么参考512890？

012708（东方红红利低波A）是指数基金，跟踪931446（东证红利低波指数）。512890（红利低波ETF）也跟踪同一指数，流动性更好，数据更及时。所以用512890的行情数据来替代参考，实际操作中买入/卖出的是012708。

❓ 价值100ETF（159263）上市时间短，MA250准吗？

159263于2025年7月上市，目前只有约205个交易日数据，不足250天。策略会自动检测：数据不足250天时，MA250标记为"参考值"，同时用更短均线作为辅助判断，不会影响定投建议的准确性。

❓ 纳指100 PE分位用近7年够吗？

近7年（约1650个交易日）覆盖了：2018年关税战、2020年疫情、2022年加息周期、2023年AI爆发。包含了多种市场环境下的PE分布，足够用于分位计算。数据太少会失真，太多会稀释近期状态。

❓ 如果三只标的都触发减仓，怎么办？

按策略分别执行，互不影响。每只标的的冷却期独立计算。减仓比例也是针对每只标的的动态仓（30%部分）分别操作，不是整体仓位一刀切。

❓ 数据是实时更新的吗？

数据每30分钟自动更新一次（GitHub Actions定时任务）。A股数据更新时间为交易日9:30-15:00，纳指100数据更新时间为北京时间21:00-次日5:00。网页也每30分钟自动刷新数据。
