  • 回补 强势区+RSI 59-63 → 接回2/3（30天冷却）
  • 回补 强势区+RSI<50 → 接回全部（30天冷却）
  • 铁律 弱势区不减仓、不回补，只按表定投
策略规则 · PE+回撤（纳指100）

月度定投建议

PE近7年分位估值基础定投深度回调(>15%)
≥85%极度高估暂停仍暂停
70-84%高估0.5份0.5份不加仓
50-69%合理1份+0.5份
30-49%低估1.8份+0.5份
<30%极度低估2.5份封顶维持2.5份

减仓/回补规则

  • 减仓 PE≥70%+回撤<8%+冷却外 → 卖出15%/30%
  • 回补 PE 40-50% → 接回2/3（60天冷却）
  • 回补 PE<40% → 接回全部（60天冷却）
  • 铁律 PE<70%不减仓；回撤>15%永不减仓
  • 底仓 剩余≥历史最高40%

每月21日执行顺序（三标的统一）

  • 1. 减仓卖出 → 2. 回补买入 → 3. 月度定投
  • 减仓后当月不再回补
  • 冷却期内跳过回补

统一铁则

  • 只用3年以上闲钱，无杠杆不融资
  • 每月只操作1次，杜绝频繁交易
  • 长期持有，低位多买 > 高位少买
  • 年底再平衡至 40%/40%/20%
