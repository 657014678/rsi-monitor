月度定投建议

RSI区间状态强势区弱势区
≥72极度过热暂停暂停
65-71偏热0.3份0.3份
55-64中性偏热0.6份0.6份
48-54中性合理1份1份
40-47偏冷低估1.5份1.2份
<40极度低估2份1.5份

减仓规则 仅强势区

  • RSI 65-71 → 卖出 15%
  • RSI ≥72 → 卖出 30%
  • 触发条件：强势区 + RSI≥65 + 不在冷却期
  • 底仓铁律：持仓不低于历史最高持仓的 50%

回补规则 仅强势区

  • 标准 RSI 59-63 → 接回已减仓 2/3（触发30天冷却）
  • 标准 RSI <50 → 接回全部剩余（触发30天冷却）
  • 豁免 减仓后90天内RSI从未≤59且价格≥MA250 → 接回1/3
  • 冷却期30天内不执行任何回补操作

弱势区铁律

  • 不卖 弱势区永远不减仓
  • 不补 弱势区不执行回补
  • 只投 弱势区按定投表正常投入

操作日流程（每月21日）

  • 执行顺序：减仓 → 回补 → 定投
  • 减仓后当月不再回补（避免反复操作）
  • 冷却期内跳过回补步骤

铁则

  • 只用闲钱投资，绝不上杠杆
  • 每月只操作1次，杜绝频繁交易
  • 长期持有，低位多买 > 高位少买
