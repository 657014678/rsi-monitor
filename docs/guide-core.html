🎯 核心逻辑（一句话）

低位多买，高位少买，强势区减仓，弱势区死扛

用数学纪律代替情绪决策，用时间换空间。不预测市场，只应对市场。

⚙️ 执行规范

操作顺序（每月21日）：① 先处理减仓/回补（若有）；② 再执行定投。减仓/回补获得的现金可用于当月的定投操作，确保资金效率。

减仓资金管理：减仓卖出获得的现金转入货币基金或短债基金暂存，作为后续定投/回补的备用资金，避免在活期账户闲置。

冷却期互锁规则：各标的独立追踪冷却期。冷却期只限制同类型操作——减仓冷却期（30天）内不能再减仓，但不影响定投和回补；回补同理，回补后重新计算的30天仅限制下次回补，不影响减仓。纳指冷却期60天，规则相同。

年底再平衡（12月21日）：每年12月21日检查三标的实际配置比例。若偏离目标权重超过5个百分点，通过卖出超配标的+买入低配标的方式恢复至40/40/20。再平衡操作不受冷却期约束（标记为"再平衡"而非"减仓"），再平衡后重置各标的冷却期计时。

02📈 A股策略：RSI21 + MA250
适用：红利低波 · 价值100 · 本月定投份数由此决定

指标说明

指标含义作用
RSI2121日相对强弱指数判断"超买/超卖"
MA250250日均线（≈年线）判断"强势/弱势区"

为什么是RSI21？与月度定投节奏匹配，过滤短期噪音。

市场分区规则

分区条件可执行操作
强势区价格 ≥ MA250定投 + 减仓 + 回补
弱势区价格 < MA250只定投，不减仓，不回补

铁则：弱势区不减仓

  • 浮亏卖出 = 真亏 + 确认亏损
  • 红利/价值100的成分股都是盈利稳定的好公司，下跌是暂时的
  • 弱势区只做一件事：按表定投，收集便宜筹码
📌 长期策略视角：弱势区是积累筹码的黄金窗口。价格低于MA250意味着市场情绪悲观，但好公司不会一直便宜。历史回测表明，在弱势区坚持定投可有效拉低平均持仓成本，当市场重回强势区时，低位筹码的收益率远高于追高买入。弱势区平均持续数月至一年，熬过去就是丰收季。不预测最低点，只做持续买入。

  1. 处于强势区（价格 ≥ MA250）
  2. RSI21 ≥ 65（市场偏热）
  3. 不在30天冷却期内
RSI21减仓比例
65–71减仓 15%（动态仓部分）

减仓对象：仅动态仓（30%部分），核心仓（70%）永不卖出。

A股回补规则

触发条件（全部满足才可以回补）：
  1. 处于强势区
  2. 之前有减仓操作 + 减仓日期在30天冷却期外
RSI21回补比例
59–63接回已减仓的 2/3
<50接回全部剩余

回补后重新计算30天冷却期。

⚠️ 快速下跌盲区：当RSI从高位（如72+）快速跌至50以下时，虽然满足"全部回补"条件，但市场可能仍处于急跌通道中，一次性回补全部减仓份额存在继续被套的风险。建议确认：① RSI在50以下至少停留5个交易日企稳；② 价格止跌回升（如连续2日收阳）后再执行回补。若仍处于急跌状态，可考虑分批回补：先回补1/3，确认站稳后再接剩余。

⚠️ 执行说明

减仓基数：减仓比例（15%/30%）以当前持仓市值为基数。例如持仓10万元，RSI≥72减仓30%，即卖出3万元对应份额。减仓份额从动态仓（30%部分）中扣除，核心仓（70%）不动。若动态仓不足，最多减到动态仓清空。

回补基数：以最近一次减仓的卖出市值为基数。例如上次减仓卖出3000元对应份额，回补2/3即买入2000元。回补后的份额归入动态仓。

冷却期管理：减仓后务必记录日期。A股冷却30天，纳指冷却60天。建议在手机备忘录或日历标注下次可操作日期。

梯度边界处理（准暂停区间）：当RSI落在梯度表的分界线附近时（如65/71/52），以达到即触发为原则——RSI≤65触发减仓线即认为达到65档，无需纠结"差0.5"的问题。特别注意RSI 71-74区间是准暂停区域：71-71.9仍然执行0.3份定投，但需高度警觉市场过热；一旦RSI触及≥72即进入暂停定投+减仓30%的区间。若RSI在此区间徘徊两个月，建议手动审视是否过度集中。

03🇺🇸 美股策略：PE分位 + 回撤
适用：纳指100（159696）· 估值+回调双维度判断

为什么纳指100不用RSI+MA250？

  • 纳指100长期牛市，MA250几乎永远在价格下方 → 弱势区信号极少
  • 美股科技股波动大，RSI频繁超买超卖 → 信号噪声太大
  • PE分位直接看"贵不贵"，回撤直接看"打折没" → 更直观，更适合纳指

指标说明

指标含义数据来源
PE分位当前PE(TTM)在近7年中的排名百分位乐咕乐股
高点回撤(当前价-阶段最高价)/阶段最高价新浪财经
📌 PE数据局限性（必读）：纳指PE分位基于乐咕乐股约7年历史数据（约1650条）。这意味着：① 仅覆盖一轮完整牛熊周期，极端估值区间的参考意义有限（例如2020年疫情底可能不在数据覆盖范围内）；② 宏观环境（利率/通胀/产业周期）变化会使历史分位"失真"——2020-2021年的零利率环境下的高PE≠泡沫；③ PE分位锚定过去而非未来，高PE不一定是卖出信号，低PE也不一定是买入信号。建议将PE分位作为参考而非决策唯一依据，与回撤数据交叉验证。

铁则：深度回调（回撤>15%）永不减仓

  • 深度回调时即使PE高估，也只暂停定投，绝不卖出
  • 历史数据：纳指100深度回调后6-12个月内多能收复失地

纳指减仓规则

触发条件（全部满足）：
  1. PE分位 ≥ 70%
  2. 回撤 < 8%（正常波动，非深度回调）
  3. 不在60天冷却期内
PE分位减仓比例
70–84%减仓 15%
≥85%减仓 30%

纳指冷却期60天（比A股长），因为美股波动大，避免频繁操作。

纳指回补规则

触发条件（全部满足）：
  1. PE分位 < 70%（退出高估区）
  2. 回撤 < 15%（不在深度回调中）
  3. 之前有减仓 + 减仓日期在60天冷却期外
PE分位回补比例
40–50%接回已减仓的 2/3
<40%接回全部剩余

⚠️ 执行说明

深度回调加码规则：当回撤 > 15% 时，在 PE 分位基础定投份数上额外 +0.5 份。例如：PE合理区(1份)+深度回调 → 1.5份；PE低估区(1.8份)+深度回调 → 2.3份。极度低估区(2.5份封顶)，不加深度加码，但回撤>15%时永不减仓。

减仓基数：与A股规则相同——以当前持仓市值为基数，从动态仓（30%部分）扣除。

回补基数：以最近一次纳指减仓的卖出市值为基数。接回2/3即买入减仓市值的2/3对应的份额。

冷却期管理：纳指冷却期60天，比A股长。减仓后务必在日历标注下次可操作日期。
