02📈 A股策略：RSI21 + MA250
适用：红利低波 · 价值100 · 本月定投份数由此决定

指标说明

指标含义作用
RSI2121日相对强弱指数判断"超买/超卖"
MA250250日均线（≈年线）判断"强势/弱势区"

为什么是RSI21？与月度定投节奏匹配，过滤短期噪音。

市场分区规则

分区条件可执行操作
强势区价格 ≥ MA250定投 + 减仓 + 回补
弱势区价格 < MA250只定投，不减仓，不回补

铁则：弱势区不减仓

  • 浮亏卖出 = 真亏 + 确认亏损
  • 红利/价值100的成分股都是盈利稳定的好公司，下跌是暂时的
  • 弱势区只做一件事：按表定投，收集便宜筹码

A股减仓规则

触发条件（全部满足才可以减）：
  1. 处于强势区（价格 ≥ MA250）
  2. RSI21 ≥ 65（市场偏热）
  3. 不在30天冷却期内
RSI21减仓比例
65–71减仓 15%（动态仓部分）
≥72减仓 30%（动态仓部分）

减仓对象：仅动态仓（30%部分），核心仓（70%）永不卖出。

A股回补规则

触发条件（全部满足才可以回补）：
  1. 处于强势区
  2. 之前有减仓操作 + 减仓日期在30天冷却期外
RSI21回补比例
59–63接回已减仓的 2/3
<50接回全部剩余

回补后重新计算30天冷却期。

03🇺🇸 美股策略：PE分位 + 回撤
适用：纳指100（159941）· 估值+回调双维度判断

为什么纳指100不用RSI+MA250？

  • 纳指100长期牛市，MA250几乎永远在价格下方 → 弱势区信号极少
  • 美股科技股波动大，RSI频繁超买超卖 → 信号噪声太大
  • PE分位直接看"贵不贵"，回撤直接看"打折没" → 更直观，更适合纳指

指标说明

指标含义数据来源
PE分位当前PE(TTM)在近7年中的排名百分位乐咕乐股
高点回撤(当前价-阶段最高价)/阶段最高价新浪财经

PE分位估值状态定投动作

30–49%低估1.8份
50–69%合理1.0份（基准）
70–84%高估0.5份
≥85%极度高估暂停定投
<30% 极度低估
30-49% 低估
50-69% 合理
70-84% 高估
≥85% 极度高估

回撤分区

回撤幅度分区对操作的影响
<8%正常波动按PE分位正常执行
8–15%中级回调可适度加码
>15%深度回调永不减仓！可大幅加码
<8% 正常波动
8-15% 中级回调
>15% 深度回调 ⚠️

铁则：深度回调（回撤>15%）永不减仓

  • 深度回调时即使PE高估，也只暂停定投，绝不卖出
  • 历史数据：纳指100深度回调后6-12个月内多能收复失地

纳指减仓规则

触发条件（全部满足）：
  1. PE分位 ≥ 70%
  2. 回撤 < 8%（正常波动，非深度回调）
  3. 不在60天冷却期内
PE分位减仓比例
70–84%减仓 15%
≥85%减仓 30%

纳指冷却期60天（比A股长），因为美股波动大，避免频繁操作。

纳指回补规则

触发条件（全部满足）：
  1. PE分位 < 70%（退出高估区）
  2. 回撤 < 15%（不在深度回调中）
  3. 之前有减仓 + 减仓日期在60天冷却期外
PE分位回补比例
40–50%接回已减仓的 2/3
<40%接回全部剩余
