# 价值投资控制台

三标的定投决策辅助系统：RSI21+MA250（A股价值） + PE分位+回撤（美股科技）

| 025497 易方达价值100A | 国证价值100 (980081) | 159263 价值100ETF | 40% | RSI21+MA250 |

## 策略 V6 · 三标的统一规则

### 统一设定
- 定投操作日：**每月21日**（遇节假日顺延）
- 资金权重固定：40% / 40% / 20%
- 年底再平衡回标准权重
- 铁则：3年以上闲钱、无杠杆、每月1次、不盯盘

---

### 红利低波 / 价值100（RSI21+MA250）

**市场分区**
- 强势区：价格 ≥ MA250 → 可减仓/回补/定投
- 弱势区：价格 < MA250 → 只定投，不减仓不回补

**月度定投倍率**

| RSI21区间 | 状态 | 强势区 | 弱势区 |
|-----------|------|--------|--------|
| ≥72 | 极度过热 | 暂停 | 暂停 |
| 65-71 | 偏热 | 0.3份 | 0.3份 |
| 55-64 | 中性偏热 | 0.6份 | 0.6份 |
| 48-54 | 中性合理 | 1份 | 1份 |
| 40-47 | 偏冷低估 | 1.5份 | 1.2份 |
| <40 | 极度低估 | 2份 | 1.5份 |

**减仓**：强势区+RSI≥65+冷却外 → 卖出15%/30%，底仓≥50%
**回补**：强势区 RSI 59-63→2/3 / RSI<50→全部，30天冷却
**弱势区铁律**：不卖、不补、只投

---

### 纳斯达克100（PE分位+回撤）

**双市场分区**
- 估值分区：极度低估(<30%) / 低估(30-49%) / 合理(50-69%) / 高估(70-84%) / 极度高估(≥85%)
- 回撤分区：正常(<8%) / 中级(8-15%) / 深度(>15%)

**月度定投倍率**

| PE近7年分位 | 估值 | 基础定投 | 深度回调(>15%) |
|-------------|------|----------|----------------|
| ≥85% | 极度高估 | 暂停 | 仍暂停 |
| 70-84% | 高估 | 0.5份 | 不加仓 |
| 50-69% | 合理 | 1份 | +0.5份 |
| 30-49% | 低估 | 1.8份 | +0.5份 |
| <30% | 极度低估 | 2.5份封顶 | 维持2.5份 |

**减仓**：PE≥70%+回撤<8%+冷却外 → 卖出15%/30%，底仓≥40%
**回补**：PE 40-50%→2/3 / PE<40%→全部，60天冷却
**深度回调铁律**：回撤>15%永不减仓
**专属铁则**：5年以上闲钱、总仓位不超20%、永不止损

---

### 每月21日统一执行顺序
1. 减仓卖出 → 2. 回补买入 → 3. 月度定投

## 技术架构

| 组件 | 方案 |
|------|------|
| 前端 | 纯静态 HTML（深色主题 · 手机优先 · 卡片式垂直流） |
| 数据引擎 | Python (AKShare + yfinance + 腾讯财经 + 乐咕PE + 新浪指数) |
| A股数据源 | Yahoo → 腾讯财经 → 东方财富(AKShare) → 新浪 |
| 纳指PE数据 | 乐咕乐股 stock_market_pe_lg（近7年历史PE TTM） |
| 纳指指数价格 | 新浪财经 index_us_stock_sina (.NDX) |
| 部署 | GitHub Pages |
| 自动更新 | GitHub Actions |

### MA250参考值
159263上市不足250天，使用 min_periods=120 计算MA250参考值，前端标注"参考"。

### PE数据说明
纳指100 PE数据来自乐咕乐股，历史约7年（2019年7月起），计算近7年分位（非10年），随数据积累自动扩展。
