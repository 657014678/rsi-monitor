## 📐 V7 策略速览

### 🅰️ A股双将（红利低波 + 价值100）

**核心逻辑**：RSI21 定力度，MA250 定能不能卖。

**定投 — 5级梯度（只看 RSI，不分强弱区）**

| RSI | 金额 | 翻译成人话 |
|-----|------|-----------|
| <40 | **2.0 份** 🟢 | 极度恐惧？冲！ |
| 40-49 | **1.5 份** 🟢 | 便宜，多买点 |
| 50-59 | **1.0 份** | 合理，正常投 |
| 60-69 | **0.5 份** 🟡 | 有点贵了，少买 |
| ≥70 | **暂停** 🔴 | 过热，钱先留着 |

**减仓条件**：强势区 + RSI≥65  
→ 15%（65-71） / 30%（≥72），跨月即冷却

**回补条件**：强势区  
→ RSI 50-64 接回 2/3，<50 接回全部，跨月即冷却

**弱势区铁律**：只定投、不减仓、不回补 <- 就这么简单

---

### 🅱️ 纳指100（PE分位+回撤）

**核心逻辑**：PE 分位看贵不贵，回撤看摔得狠不狠。

**定投 — 5级梯度 + 深度回调加成**

| PE分位 | 估值 | 基础定投 | 深度回调加码 |
|--------|------|---------|-------------|
| ≥85% | 极度高估 🔴 | 暂停 | — |
| 70-84% | 高估 🟡 | 0.5 份 | — |
| 50-69% | 合理 | 1.0 份 | +0.5 份 |
| 30-49% | 低估 🟢 | 1.8 份 | +0.5 份 |
| <30% | 极度低估 🟢 | **2.5 份封顶** | +0.5 份 |

**减仓**：PE≥70% + 回撤<8% + 已跨月 → 15% / 30%  
**回补**：PE<70% 即可，跨月即冷却（去除回撤限制）  
**深度回调铁律**：回撤>15% 永不减仓

## 🔄 数据更新机制（双轨并行，妈妈再也不用担心我忘了更新）

### 🚄 高速轨 — 每 30 分钟实时行情
交易时段内，**GitHub Actions** 每 30 分钟自动跑一次：  
→ 拉 ETF 实时价 → 更新页面 → 部署  
**不开电脑也能更新**，开会、睡觉、摸鱼都不耽误。

### 🐢 慢速轨 — 每日 15:30 完整快照
收盘后用 **AKShare** 拉指数日K线 → 算 RSI、MA250、PE 分位
→ 生成完整数据文件 → git push → GitHub Pages 自动部署

### 🏠 本地保底
Windows 定时任务（`setup_task.ps1` 一键安装）  
电脑开着时本地也跑，双重保险。

## 🏗️ 技术架构

| 组件 | 方案 |
|------|------|
| 前端 | 纯静态 HTML（深色 · 手机优先 · 卡片流） |
| 数据引擎 | Python AKShare |
| A股指数数据 | 中证指数官网 (CSIndex) + 国证指数官网 (CNI) |
| 纳指PE | 乐咕乐股 stock_market_pe_lg |
| 纳指价格 | 新浪财经 .NDX |
| ETF实时行情 | 腾讯财经 API（每30分钟） |
| 回测引擎 | Python 三标的同步回测（含再平衡） |
| 部署 | GitHub Pages |
| 外壳 | GitHub Actions + Windows 定时任务 |

### 文件结构

```
├── .github/workflows/
│   ├── realtime_update.yml    # 🚄 每30分钟实时行情（主力）
│   └── pages.yml              # GitHub Pages 部署
├── docs/
│   ├── index.html             # 🏠 主控制台（每日一看）
│   ├── guide-*.html           # 📖 指南系列（策略/规则/心法/回测）
│   ├── style-guide.css        # 🎨 全局样式
│   ├── data.json              # 📊 合并数据（前端加载这个）
│   ├── h30269.json            # 红利低波指数数据
│   ├── 980081.json            # 价值100指数数据
│   └── nd100.json             # 纳指100数据（PE+价格）
├── backtest_dingtou.py        # 📊 三标的定投回测
├── daily_update.py            # ⭐ 每日收盘后完整更新（主力）
├── update_realtime.py          # 🚄 每30分钟轻量行情更新
├── setup_task.ps1             # Windows 定时任务安装脚本
├── requirements.txt           # Python 依赖
└── README.md                  # 就这个文件 👋
```

### 数据流

```

   https://free7566.github.io/rsi-monitor/
```

## 🚀 本地部署

```bash
# 安装依赖
pip install -r requirements.txt

# 手动跑一次，验证数据
python daily_update.py --no-push

# 查看生成的 JSON
ls docs/
```

### 每日自动更新

```bash
# 管理员 PowerShell → 项目目录
.\setup_task.ps1
```

创建两个定时任务：
- **RSI Monitor Daily Update** → 每日 15:30（收盘快照）
- **RSI Monitor Realtime Update** → 每 30 分钟（实时行情）

### 手动更新

```bash
python daily_update.py            # 更新 + git push
python daily_update.py --no-push  # 只更新不推送（先看看效果）
```
