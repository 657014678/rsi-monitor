# 红利低波长期投资控制台

基于RSI21+MA250非对称仓位策略的红利低波ETF长期定投监控平台

## 功能特点

- 📊 实时监测ETF价格、RSI21、MA250等关键指标
- 🎯 V4非对称仓位策略（低位重仓、高位轻仓）
- 💡 7级仓位动态建议（20%~300%）
- 🚦 三级风险信号提示（绿/黄/红）
- 📈 交互式K线图（含MA250均线）
- 🤖 自动化数据更新（GitHub Actions）
- 📋 策略回测可视化（7年+历史数据）

## 投资策略 V4

### 核心理念
红利低波ETF长期向上，回调是买入机会而非风险。策略以**低位多买**为核心，通过非对称仓位梯度实现超额收益。

### 7级仓位梯度

| RSI区间 | 状态 | 仓位 | 说明 |
|---------|------|------|------|
| < 28 | 极度恐慌 | 300% | 翻倍加仓 |
| 28-40 | 低估 | 200% | 大幅加仓 |
| 40-48 | 偏冷 | 140% | 适度加仓 |
| 48-55 | 正常偏冷 | 90% | 接近全仓 |
| 55-65 | 正常偏热 | 60% | 适度减仓 |
| 65-72 | 偏热 | 40% | 明显减仓 |
| > 72 | 高估 | 20% | 最低仓位 |

### 加成机制
- MA250下方 + RSI<40：额外+30%加仓
- 连续2月低位：额外+30%
- 连续3月低位：额外+60%
- 连续4月+低位：额外+100%

### 保守模式
- 触发：连续15天跌破MA250 + MA下行 → 仓位降至50%
- 恢复：连续5天站上MA250 + MA趋平/上行 → 恢复正常
